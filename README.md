# weather-game

☀️ 線上天氣預報遊戲 - 挑戰你的天氣預測能力！

## 功能特色

- 🌍 **城市選擇** - 12個台灣城市可選
- 🌡️ **溫度挑戰** - 滑桿選擇溫度，精確命中獲得高分
- 🌤️ **天氣預測** - 選擇正確的天氣狀況
- 💰 **積分系統** - 精確度越高，分數越多
- 🔥 **連勝系統** - 連續答對可獲得額外獎勵
- 💡 **提示功能** - 花費積分獲得提示
- 📊 **排行榜** - localStorage 儲存積分
- 📱 **響應式設計** - 支援手機和桌面

## 遊戲規則

### 溫度題目
- 完全命中（差0°C）：100分
- 非常接近（差1-2°C）：70分
- 接近（差3-5°C）：40分
- 差較多（差6°C以上）：10分

### 天氣題目
- 完全正確：100分
- 錯誤：0分

### 提示系統
- 使用提示：最終積分扣50分

## 如何遊玩

1. 選擇一個城市
2. 回答10道題目（溫度或天氣）
3. 每題結束後可查看答案
4. 遊戲結束後儲存分數到排行榜
